 ゲーム概要
	タイトル         :  棒人間のキセキ
	プレイ人数       :  1人
	プラットフォーム :  PC(Windows/DirectXVer9.0)
	制作環境         :  Windows10/DirectXVer9.0
	制作期間         :  H30年12月〜H31年2月(約３ヶ月)
	制作人数         :  1人
	使用言語         :  C++
	使用ソフト       :  Visual Studio2015 / Adobe Photoshop CC 2018 / MetasequoiaLER3.0
	こだわりポイント :  草の処理 / 攻撃時のエフェクト / ボスのAI制御
 
 主な操作方法
	****  キーボード操作  ****
		---  メニュー  ---
			決定               :  [ ENTER ]キー
			項目選択           :  [ W ] [ S ]キー

		---  プレイヤー  ---
			移動               :  [ W ] [ A ] [ S ] [ D ]キー
			カメラ操作         :  [ ↑ ] [ ← ]  [ ↓ ]  [ → ]キー
			視点リセット       :  [ SHIFT(左側) ]キー
			攻撃               :  [ SPACE ]キー
			ジャンプ           :  [ ENTER ]キー
			オラクルアタック   :  [ B ]キー
			ロックオン         :  [ E ]キー
			軌跡斬切り替え     :  [ R ]キー
			分身を増やす       :  [ N ]キー
			分身を減らす       :  [ M ]キー

	****  コントローラー操作  ****
	( ELECOM社製 JC-U3613MBD ( DirectInput )で動作確認済み )
		---  メニュー  ---
			決定               :  Bボタン
			項目選択           :  左スティック入力
			
		---  プレイヤー  ---
			移動               :  左スティック入力 / 十字キー入力
			カメラ操作         :  右スティック入力
			視点リセット       :  右スティック押し込み
			攻撃               :  Bボタン
			ジャンプ           :  Aボタン
			オラクルアタック   :  Yボタン
			ロックオン         :  LBボタン
			軌跡斬切り替え     :  RBボタン
			分身を増やす       :  LTボタン
			分身を減らす       :  RTボタン
 
 簡単な紹介
 	「キセキ」という表現を活かして作成したゲームです。
